# Ejercicios Kotlin
## 1. Ejercicios en Kotlin | Hallar el área de un triángulo
```kotlin
    fun main (args: Array<String>) {
    
        var base = 5;
        var altura = 6;
        var area = base * altura / 2
    
        println("El área del triángulo es igual a $area")
    }

 ```
